# CTF Writeups

### How to run scripts in this project.
#### Node.js scripts
1. Setup [Node.js LTS](https://nodejs.org/en/) environment.
2. Clone the repository

    ```sh
    git clone https://github.com/imyelo/ctf-writeups.git
    ```

3. Install Node.js dependencies:

    ```sh
    npx lerna bootstrap
    ```
